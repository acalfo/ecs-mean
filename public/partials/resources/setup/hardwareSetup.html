Hardware Setup

NOTE: Arduino Driver installation requires administrator privileges.

  1. Download the MUzECS software here.

  2. Unzip muzecs_software.zip and remember where you choose to save the unzipped folder.

  3. Connect the Arduino Leonardo to the computer with a micro USB cable. You should see an item appear in the taskbar on the bottom of the screen. Wait for this to finish and go away.

  4. When this is finished, open the Device Manager. This can be done by click the windows button in the bottom of the screen (or pressing the windows key on the keyboard) and typing "Device Manager" into the search bar.

  5. You should see the Arduino Leonardo listed under "Other Devices". Right click on "Arduino Leonardo" and select "Update Driver Software". A window should appear presenting you with two options: Find drivers online or browse your computer for driver software.

  6. Select the "Browse my computer for driver software" option.

  7. Click "Browse" and find the folder labeled "drivers" inside of the folder that you unzipped in step 2. After selecting the "drivers" folder, make sure that "Include subfolders" is checked in the window that appears afterward, then click Next.

  8. A popup will appear that asks if you would like to install this device software. Click "Install".

  9. The Arduino drivers should be installed at this point. Verify this by verifying that "Arduino Leonardo" is now listed under "Ports (COM & LPT)" instead of "Other Devices".
    To test, run the Arduino application (arduino.exe) from the folder you unzipped in step 2. A Windows firewall alert will appear regarding Java requesting access to the internet. Click "Cancel" to dismiss this window. If you allow access then you will likely receive a popup about updating Arduino each time you open the program and the MUzECS team has not yet verified that this update process works correctly, so it's best to just disallow it for now.

  10. Click on Tools in the menu and select the Arduino Leonardo as your Board and the appropriate COM port for the device.

    • If you do not see "COM# (Arduino Leonardo)" as an option under Tools > Port, then we must find out which port number to select. To do this, open the Device Manager just as you did before. Find "Arduino Leonardo" under "Ports (COM & LPT)". The COM port will be listed alongside "Arduino Leonardo".
